1. Скачивание zrok и его настройка

1. Заходите на сайт zrok.io и скачиваете программу ИЛИ скачиваете последнюю версию с github выбрав СВОЮ ОС. При необходимости зарегистрируетесь. Расположите папку с zrok где вам удобно.

P.S При дальнейшей настройке понадобится почта, которая будет у хоста-Evereast, также потребуется регистрация на сайте и взятие токена с сайта

2. Заходите на сайт zrok.api и копируете свой токен как на фото снизу. При необходимости регистрируетесь/входите в аккаунт

3. Будем запускать zrok. Для этого введите в проводнике где папка с zrok.exe - cmd - как на фото снизу, должна открыться консоль

После пишем в неё zrok enable 'ваш токен', пример - zrok enable O8abVnfaE1

zrok enable

Обязательно! На этом этапе уже кидаете свою почту, на которой зарегистрирован аккаунт zrok

Вы не сможете создать туннель в zrok если у хоста не будет вашей почты

2. Подключение туннеля для zrok

1. Есть несколько возможностей использовать команды для подключения:

1.1 Не предпочтительный способ через запуск cmd и ввод команды (для Minecraft) (Работает в терминале на Linux):

zrok.exe access private ever --bind 127.0.0.1:25565

И команда для войс-чата:

zrok.exe access private everudp --bind 127.0.0.1:25565
P.S. При двух подключениях нужно писать в 2 разные cmd!!!

1.2 Наиболее легкий способ для Windows:

Скачиваете .bat файл с такой командой, или создайте свой батник с такой командой

Батник должен быть в папке с zrok.exe для корректной работы!

@echo off
start cmd /k "zrok.exe access private ever --bind 127.0.0.1:25565 && pause"
start cmd /k "zrok.exe access private everudp --bind 127.0.0.1:25565 && pause"

Если вы захотите запустить zrok просто запускайте .bat файл и 2 подключения будут автоматически созданы

FAQ

Когда запускать zrok?

-Тогда, когда Evereast напишет что он запустил сервер

Что будет если запустить zrok раньше?

-Не произойдет подключение туннеля и надо будет перезапускать, так так zrok не проверят есть ли соединение и не будет пробовать переподключаться

Почему я должен включать неизвестные программы у себя?

-Все программы open-sourse, вы сами можете просмотреть код, проверить по virustotal и тп, а нужен zrok нам для соединения, так как у нас нету белого ip
